**Skill ID:** `file-manager`
**版本:** 1.0.0

## 描述

智能文件管理器，支持文件扫描、自动归类、批量重命名、复制移动、查找重复文件、清理整理桌面等功能。零删除、零篡改，安全可靠。

---

## 使用场景

- 整理混乱的桌面
- 批量重命名文件
- 按类型归类文件
- 查找指定类型的文件
- 清理临时文件和缓存
- 查找重复文件

## 安全原则

1. **零删除**：不会自动删除任何文件
2. **零篡改**：不修改文件内容
3. **可回撤**：所有操作记录日志，支持一键回撤
4. **确认执行**：危险操作会先询问用户
