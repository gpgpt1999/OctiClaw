# Web Search Skill

**Skill ID:** `web-search`
**版本:** 1.0.0
**图标:** 🔍

---

## 描述

集成多搜索引擎的网页搜索工具，支持实时联网搜索、关键词提取、新闻聚合。覆盖 Google、Bing、百度等国内外主流搜索引擎。

---

## 使用场景

- 查询实时信息（股价、天气、新闻）
- 验证事实和数据
- 搜索教程、资源和参考资料
- 了解某个领域的最新动态
- 寻找答案和解决方案

---

## 触发关键词

- "搜一下"、"搜索"、"查一下"
- "网上有什么关于..."
- "search for"、"look up"
- "最近/最新的...是什么"

## 基本用法

### 搜索网页

```
搜一下 Python 最新版本特性
```

```
搜索 2024 年诺贝尔物理学奖得主
```

### 指定搜索引擎

```
用 Google 搜索 AI 发展趋势
```

```
用百度搜索 国内新能源车企排名
```

### 搜索新闻

```
搜索最近关于 AI 的新闻
```

```
查一下今天的科技新闻
```

### 高级搜索

```
搜索 site:github.com openai whisper
```

## 工具调用

此 Skill 使用内置 `web_search` 工具：

```json
{
  "tool": "web_search",
  "query": "搜索关键词",
  "count": 10,
  "country": "US",
  "language": "zh",
  "freshness": "week"
}
```

### 参数说明

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| query | string | ✅ | 搜索关键词 |
| count | number | ❌ | 返回结果数量（1-10），默认 10 |
| country | string | ❌ | 国家代码，如 US、CN |
| language | string | ❌ | 语言代码，如 zh、en |
| freshness | string | ❌ | 时间过滤：day/week/month/year |
