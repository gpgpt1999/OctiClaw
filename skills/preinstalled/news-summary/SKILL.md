# News Summary Skill

**Skill ID:** `news-summary`
**版本:** 1.0.0
**图标:** 📰

---

## 描述

多源新闻聚合工具，自动从 RSS 订阅源抓取科技、财经、国际等分类新闻，生成智能摘要。支持语音播报新闻简报。

---

## 使用场景

- 每日新闻简报
- 行业动态追踪
- 热点事件了解
- 科技资讯获取
- 语音新闻播报

---

## 工具调用

此 Skill 使用 `web_fetch` 或调用 RSS API：

```json
{
  "tool": "web_fetch",
  "url": "https://rss.example.com/tech"
}
```

支持语音播报时使用 `tts` 工具：
```json
{
  "tool": "tts",
  "text": "今日新闻摘要..."
}
```
