# Image Generation Skill

**Skill ID:** `image-gen`
**版本:** 1.0.0
**图标:** 🎨

---

## 描述

AI 驱动的图像生成工具，支持文生图（Text-to-Image）、图生图（Image-to-Image）、图片编辑、海报设计、Logo 制作等。让创意可视化，一键生成。

---

## 使用场景

- 创作配图
- 设计海报/Banner
- 生成 Logo 概念
- 图片风格迁移
- 产品展示图
- 社交媒体配图

## 工具调用

此 Skill 使用图像生成 API：

```json
{
  "tool": "canvas",
  "action": "present",
  "javaScript": "generateImage(prompt)"
}
```

或调用 DALL·E / Stable Diffusion API。

---

## 参数说明

| 参数 | 类型 | 说明 |
|------|------|------|
| prompt | string | 图像描述词 |
| style | string | 风格：写实/卡通/水彩/科技等 |
| size | string | 尺寸：1:1/16:9/9:16 |
| quality | string | 质量：standard/high |

---

## 依赖说明

| 依赖 | 是否必需 | 说明 |
|------|----------|------|
| 图像生成 API | ⚠️ | 需要配置 API Key |
| Canvas 工具 | ❌ | 用于展示结果 |
