## 工具调用

此 Skill 使用 Python 执行 Excel 操作：

```python
import pandas as pd

# 读取 Excel
df = pd.read_excel("data.xlsx")

# 数据处理
df_clean = df.dropna()
df_grouped = df.groupby("category").sum()

# 导出
df_clean.to_excel("output.xlsx")
```
