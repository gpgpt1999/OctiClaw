# Weather Skill

**Skill ID:** `weather`
**版本:** 1.0.0
**图标:** 🌤️

---

## 描述

实时天气查询工具，提供当前天气、未来7天预报、穿衣指数、出行建议等贴心提醒。支持全球任意城市查询。

---

## 使用场景

- 出门前查看天气
- 安排出行计划
- 穿衣搭配建议
- 了解目的地天气
- 活动安排参考

## 工具调用

此 Skill 使用内置 `weather` 工具或调用天气 API：

```json
{
  "tool": "exec",
  "command": "curl wttr.in/Beijing?format=3"
}
```

或使用 Open-Meteo API 获取详细数据。

---

## 返回信息

| 信息 | 说明 |
|------|------|
| 当前温度 | 摄氏度 |
| 天气状况 | 晴/多云/雨/雪等 |
| 湿度 | 相对湿度百分比 |
| 风速 | km/h |
| 体感温度 | 考虑风寒效应 |
| 穿衣建议 | 根据温度推荐 |
| 出行建议 | 是否需要带伞等 |
